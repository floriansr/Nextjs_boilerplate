## Useful scripts

```
yarn test:watch
```
It will open jest console without coverage

```
yarn ts:check
```

It will check TS bugs


```
yarn ts:coverage
```

It will check TS types coverage (score atm (01/12/2020): 1498 / 1506 **99.46%**)
